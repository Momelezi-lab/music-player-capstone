# My Music Player  

A beautiful, responsive music player built with **React + Vite**, **Tailwind CSS**, and the **Deezer API**.  
Search for any song or artist and play 30-second previews instantly â€” no login, no Spotify premium needed!



## Features

- Search songs, artists, and albums in real-time  
- Beautiful responsive grid (works perfectly on mobile & desktop)  
- Play 30-second song previews with native `<audio>` controls  
- Glassmorphism cards with hover effects  
- Gradient dark background (looks like a premium app)  
- Fast & lightweight (thanks to Vite)  
- Fully styled with Tailwind CSS  

## Tech Stack

- **React 18** + **Vite** (blazing fast development)
- **Tailwind CSS** (modern utility-first styling)
- **Axios** (fetching data from Deezer API)
- **Deezer Public API** (free, no auth required): `https://api.deezer.com`
